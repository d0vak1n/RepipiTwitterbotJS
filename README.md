## Usage <a name='usage'></a>

.env is required with this template:

```

PORT="3000"

# Api Twitter acces keys v1
API_KEY="appKey"
API_SECRET="appSecret"
ACCESS_TOKEN="accessToken"
ACCESS_SECRET="accessSecret"
```
